## 6. Идея, метод, алгоритм решения задачи

Создаем директорию, затем в ней создаем файл и записываем туда фрукты, далее считаем количество строк в файле, далее ищем номера строк со словом apple, далее копируем файл в файл2, узнаем тип файла, узнаем путь до нынешней директории, перемещаем файл в ласт, удаляем файл с запросом на разрешение удаления и выводим last
